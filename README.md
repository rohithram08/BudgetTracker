# 💳 Smart Budget Tracker

A beautiful and user-friendly Java application for tracking your income and expenses with real-time balance management.

## ✨ Features

- **Add Income** ➕ - Track all your income sources
- **Add Expenses** ➖ - Record your spending
- **View Balance** 💰 - See your current balance with color-coded display
- **Transaction History** 📜 - View all transactions with timestamps
- **Input Validation** ✓ - Prevents invalid entries and overspending
- **Colorful UI** 🎨 - Enhanced terminal experience with ANSI colors and emojis
- **Error Handling** - Graceful handling of invalid inputs

## 🚀 Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher

### Installation

1. Clone the repository:
`ash
git clone https://github.com/rohithram08/BudgetTracker.git
cd BudgetTracker
`

2. Compile the program:
`ash
javac BudgetTracker.java
`

3. Run the application:
`ash
java BudgetTracker
`

## 📖 Usage

Once the program is running, you'll see a menu with the following options:

`
1. ➕ Add Income - Enter an income amount
2. ➖ Add Expense - Enter an expense amount
3. 💰 View Balance - Display current balance
4. 📜 View History - See all transactions with timestamps
5. 🚀 Exit - Close the application
`

Simply enter the number corresponding to your choice and follow the prompts.

## 📊 Example Session

`
💳 SMART BUDGET TRACKER 💳

1. Add Income: Rs 50000
2. Add Expense: Rs 5000
3. View Balance: Rs 45000
4. View History: Shows all transactions
5. Exit
`

## 💡 Tips

- All amounts should be positive numbers
- You cannot add an expense greater than your current balance
- All transactions are timestamped for your records
- Colors indicate: Green = Income/Positive, Red = Expense/Negative, Cyan = Balance Display
